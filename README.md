# Decoding-Attrition-A-Data-Driven-Revenue-Risk-Customer-Retention-Intelligence-Framework

## 📌Project Overview
An e-commerce company with 450,000 customers was experiencing a ~24.96% churn rate, putting over ₹1.1 billion in annual revenue at risk.
This project delivers a complete, end-to-end churn intelligence system, from raw CSV files to an interactive Power BI dashboard and a machine learning prediction model using a SQL-first, analytics engineering approach.

## Dataset Summary
| Table              | Rows    | Description                                      |
|--------------------|---------|--------------------------------------------------|
| customers_table.csv | 450,000 | Customer profiles + churn flag                   |
| orders_final.csv    | 450,000 | Transaction history with status & discount data  |
| activity_final.csv  | 450,000 | Behavioral engagement signals                    |
## Key Findings

- 📉 ~25% churn rate, 112,325 customers lost
-  ₹1.1 Billion+ in revenue tied to churned customers
-  Customers inactive 90+ days had 3× higher churn probability
-  High discount-dependent customers churn significantly more
-  Customers with 2+ returns show elevated churn risk (bad product experience)
-  Referral acquired customers are the most loyal; Ads acquired have the highest churn
